Manjaro Cheatsheet

I’m new to Arch based linuxes and have just decided to try Manjaro 20.0

Here is my reminder page while I get used to installing using pacman insteat of apt

Pacman

Update sources:

sudo pacman -Syy

Upgrade packages:

sudo pacman -Syu

Install package (ensuring system is up to date first to avoid breaking):

sudo pacman -Syu package_1 package_2

Install package (Don’t do this!)1:

sudo pacman -S package_1 package_2

Remove package:

sudo pacman -R package

Remove package and unused dependencies:

sudo pacman -Rs package

Remove package, unused dependencies and config:

sudo pacman -Rs package

  1. Never install a package without updating the system first. On a rolling release this can lead to an unbootable system. ↩︎
